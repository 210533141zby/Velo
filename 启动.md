如果想把它传到 GitHub，不能直接点“上传”，需要经过 三个步骤 （这是为了防止你手滑误传）：
 第一步： git add . （打包/暂存）
告诉 Git：“这些文件我改好了，准备打包。”

- . 代表当前目录下的所有文件。 第二步： git commit -m "备注信息" （盖章/提交）
告诉 Git：“确认这些修改，并给这个版本贴个标签（备注）。”

- 备注信息（如 "修复了编辑器Bug" ）非常重要，方便以后查阅。 第三步： git push （发货/推送）
告诉 Git：“把刚才打包盖章好的版本，上传到 GitHub。”
